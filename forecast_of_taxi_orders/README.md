# Прогнозирование количества заказов такси на следующий час
Компания такси собрала исторические данные о заказах такси в аэропортах. Чтобы привлекать больше водителей в период пиковой нагрузки, нужно спрогнозировать количество заказов такси на следующий час. Для такого предсказания построена модель, позволившая достичь требуемого значения метрики RMSE.
